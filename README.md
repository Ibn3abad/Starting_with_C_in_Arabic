# ابدأ مع لغة C بالعربية  
## Starting with C in Arabic

كتاب عربي شامل لتعلّم لغة **C** من الصفر، مخصص للمبتدئين الذين يريدون فهم البرمجة بعمق، وليس فقط نسخ الأكواد.

---

## 📘 عن الكتاب

هذا المشروع يحتوي على كتاب تعليمي باللغة العربية يشرح لغة C خطوة بخطوة، بدايةً من المفاهيم الأساسية مثل المتغيرات، أنواع البيانات، الشروط والحلقات، وصولاً إلى مواضيع أكثر تقدماً مثل المؤشرات، الهياكل، إدارة الذاكرة، Makefile، وغيرها.

الهدف من الكتاب هو أن يفهم القارئ:

- كيف تعمل لغة C من الداخل
- لماذا نكتب كل سطر في البرنامج
- كيف يتم التعامل مع الذاكرة
- كيف تُبنى البرامج بشكل منظم
- كيف ينتقل من مستوى المبتدئ إلى مستوى أقوى في البرمجة

---

## 👥 لمن هذا الكتاب؟

هذا الكتاب مناسب لك إذا كنت:

- مبتدئاً تماماً في البرمجة
- تريد تعلم لغة C من الصفر
- درست البرمجة سابقاً ولكنك تريد فهم الأساسيات بشكل أفضل
- مهتماً بالأنظمة المدمجة Embedded Systems
- تريد فهم الذاكرة، المؤشرات، Stack و Heap

## 🛠️ المتطلبات

لكي تطبق أمثلة الكتاب، تحتاج إلى:

- جهاز كمبيوتر
- محرر كود مثل VS Code
- مترجم C مثل GCC
- معرفة بسيطة باستخدام الطرفية Terminal

### Linux

```bash
sudo apt update
sudo apt install gcc make
```

### macOS

```bash
xcode-select --install
```

### Windows

يمكنك استخدام أحد الخيارات التالية:

- MinGW
- MSYS2
- WSL - Windows Subsystem for Linux

## ▶️ مثال سريع

```c
#include <stdio.h>

int main() {
    printf("Hello World!\n");
    return 0;
}
```

لتجميع وتشغيل البرنامج:

```bash
gcc hello.c -o hello
./hello
```
